## ⚡ Installation
  - 1️⃣ **Download**
```bash
git clone https://github.com/arm1n18/vin-decoder.git
cd vin-decoder
```
  - 2️⃣ **Install dependencies**
```bash
npm i
```
  - 3️⃣ **Run**
```bash
npm run dev
```
